Changes for Build 3

  • Removed --disable-shared option when configuring final compiler that prevented building of libgcc.so shared library and prevented iperf and other binaries from working properly. Also removed obsolete comment in build specification file.
  • Added patch to fix TLS problem with RPC state variable structure that was causing RPC to fail.

Changes for Build 2

  • Fixed syscall() patch.
  • Split up syscall() and pipe() patches into two separate files. The pipe() patch file is included, but not applied. It will be tested at a later date.
  • Added new binutils-2.17.50.0.12-999-brcm-toolchain-version.patch to denote binutils patchset version.
  • Removed binutils patch for ARM EABI.
  • Removed commented out patch lines for binutils and renumbered the patches.
  • Renamed the spec file to contain the build number.
  • Added new file Toolchain_BuildLog.gz to the info directory which contains the complete buildlog of the toolchain. This requires the command line for building to be rpmbuild -ba specfile >& buildlog. The buildlog file will then be copied and compressed.
  • Added new patch for memmove() optimized function.
  • Updated prefetch memcpy() optimized function patch.
  • Updated HTML page.
  • Bumped build date and version.

Changes for Build 1

  • Initial release with matching functionality of 2.6.12 NPTL Build 10.
  • Updated uClibc memcpy.S patch from Broadcom.
  • Added new uClibc patch to fix MIPS pipe() and syscall() functions.
  • Back-ported gprof support to work with NPTL.
  • Updated default configuration files to add gprof configuration option.
  • Removed --with-sysroot option to allow toolchain to be moved to different directory.
  • Made toolchain relocatable.
  • Added this file.
  • Bumped build date and version.
